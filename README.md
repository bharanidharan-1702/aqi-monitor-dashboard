# AQI Monitor Dashboard 🌍

## 📌 Description

This project predicts and monitors Air Quality Index (AQI) using machine learning and visualization techniques.

## 🚀 Features

* Real-time AQI monitoring
* Traffic-based prediction
* Map visualization (Leaflet)
* Machine Learning forecasting (LSTM)

## 🛠 Tech Stack

* Frontend: React + Vite + Tailwind CSS
* Backend: Spring Boot
* Database: PostgreSQL
* ML: Python

## ⚙️ Setup Instructions

1. Clone the repository
2. Install frontend dependencies
3. Run backend server
4. Start frontend

## ⚠️ Note

OSM (.pbf) file is not included due to size.
Download it manually before running the project.
